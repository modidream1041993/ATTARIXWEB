🔑 الربط بـ GitHub (مرة واحدة بس)

إزاي أجيب المفتاح؟ (خطوات بسيطة)
  1. افتح صفحة إنشاء المفتاح في GitHub (لازم تكون مسجّل دخول بحسابك modidream1041993).
  2. اكتب أي اسم (مثلاً: attarix-admin) واختار مدة انتهاء طويلة (سنة).
  3. في Repository access اختار Only select repositories واختار ATTARIXWEB بس.
  4. في Permissions → Repository permissions → Contents اختار Read and write.
  5. اضغط Generate token وانسخ المفتاح والصقه هنا فوق.
⚠️ متشاركش المفتاح ده مع حد، ومتحطوش في أي ملف بترفعه على النت.
