## 1) الرسائل التلقائية (Settings ← Business tools)

### رسالة الترحيب (Greeting message)
```
أهلاً بيك في ATTARIX 👋
نظام إدارة المحلات الذكي: كاشير + مخزون + محاسبة — يعمل بدون إنترنت وبدون اشتراك شهري.

قولنا بس:
1️⃣ نشاطك إيه؟ (سوبر ماركت / عطارة / ملابس / قطع غيار...)
2️⃣ عندك كام جهاز كاشير؟

وهنبعتلك العرض المناسب ليك فورًا 🔥
```

### رسالة خارج أوقات العمل (Away message)
```
شكرًا لتواصلك مع ATTARIX 🌙
إحنا خارج أوقات العمل حاليًا — رسالتك وصلت وهنرد عليك أول ما نبدأ الصبح إن شاء الله.
لو حابب، اكتبلنا نشاطك وعدد الأجهزة عشان نجهزلك العرض قبل ما نكلمك.
```

## 2) الردود السريعة (Quick Replies) — سجلها بالاختصارات دي

### `/سعر`
```
بخصوص السعر 💰
عندنا عرض إطلاق لفترة محدودة (باقي مقاعد قليلة) — والسعر بيختلف حسب الباقة:

🔹 الأساسية: جهاز واحد بكل المزايا — دفعة واحدة بدون اشتراك
⭐ الاحترافية: + نقل بياناتك القديمة + تدريب كامل + دعم أولوية + سنة تحديثات
☁️ السحابية: من أي جهاز وأكثر من فرع — باشتراك

قولي نشاطك وعدد الأجهزة وأقولك سعر العرض بالظبط 👌
```

### `/فرق`
```
الفرق بين الباقات باختصار:
🔹 الأساسية = البرنامج كامل بكل مزاياه على جهاز واحد + التركيب والتفعيل + دليل الاستخدام.
⭐ الاحترافية = نفس الكلام + بننقلك أصنافك وعملاءك وأرصدتك من دفاترك أو برنامجك القديم + بندرّبك انت والكاشير + دعم بأولوية + سنة تحديثات مجانية.
☁️ السحابية = تفتح من أي جهاز أو موبايل، مناسبة للفروع — باشتراك مريح.

الأغلبية بتختار الاحترافية عشان بتشتغل من أول يوم من غير مجهود 😉
```

### `/ديمو`
```
أكيد! الديمو مجاني تمامًا وبدون أي التزام 🎬
بنعمله بطريقتين:
1️⃣ أونلاين: بنتصل بيك وبتشوف البرنامج شغال قدامك على شاشتك (10 دقايق).
2️⃣ فيديو: ابعتلك فيديو 3 دقايق يوريك أهم حاجة.

تحب أنهي طريقة؟ ولو أونلاين — إمتى يناسبك النهاردة أو بكرة؟
```

### `/تركيب`
```
التركيب بسيط جدًا 🛠
- بنركّب البرنامج على جهازك (ويندوز) في أقل من 10 دقايق.
- بنظبط الطابعة والباركود سكانر معاك.
- في الباقة الاحترافية بننقل بياناتك القديمة وبندرّب الكاشير.
- وبتستلم دليل استخدام كامل بالاختصارات.

من لحظة ما تقول موافق لحد ما تبيع أول فاتورة: يوم واحد بالكتير.
```

### `/ضمان`
```
مطمّنين تمامًا 🛡
- ديمو مجاني تشوفه قبل ما تدفع مليم.
- ضمان استرجاع خلال 14 يوم لو البرنامج مش مناسب ليك.
- بياناتك على جهازك انت (مش عندنا) + نسخة احتياطية تلقائية كل يوم.
- وأي حذف بالغلط جوه البرنامج ليه زر «تراجع» يرجّعه.
```

### `/نقل`
```
نقل بياناتك مسؤوليتنا 📦
شغال بدفاتر؟ برنامج قديم؟ ملفات إكسيل؟
في الباقة الاحترافية بننقلك: الأصناف بأسعارها، العملاء بأرصدتهم، والموردين — وبتفتح البرنامج تلاقي محلك جاهز زي ما هو.
ابعتلنا عينة من بياناتك (صورة دفتر أو ملف) ونقولك هناخد وقت قد إيه.
```

## 3) سيناريو البيع الكامل (من أول رسالة للإغلاق)

**العميل:** «عايز أعرف تفاصيل البرنامج» / «بكام؟»

**أنت (خطوة 1 — تأهيل، متقولش سعر لسه):**
```
أهلاً بيك 🌟 هساعدك توصل لأنسب حاجة —
بس الأول قولي:
- نشاطك إيه بالظبط؟
- بتبيع جملة ولا قطاعي ولا الاتنين؟
- شغال حاليًا بإيه؟ (دفاتر / برنامج تاني / ولا حاجة)
```

**خطوة 2 — اربط وجعه بالميزة (اختار اللي يناسب رده):**
- قال عطارة/جملة → «يبقى هتستفيد جدًا من البيع بالكرتونة والدستة بباركود خاص، والبرنامج بيفتكر آخر سعر لكل عميل جملة عندك.»
- قال سوبر ماركت → «يبقى أهم حاجة ليك سرعة الكاشير (فاتورة في 15 ثانية) ومساعد الشراء اللي بيقولك تشتري إيه قبل ما يخلص.»
- قال بيبيع آجل → «عندنا زر واتساب جنب كل عميل مديون يبعتله تذكير مهذب بالمبلغ — فلوسك هترجع أسرع.»
- قال دفاتر → «بننقلك كل اللي في الدفاتر بنفسنا — بتفتح البرنامج تلاقي محلك جاهز.»

**خطوة 3 — قدّم الباقة والسعر:**
```
على كده أنسب حاجة ليك باقة [الاحترافية] 💎
[سعر العرض] بدل [السعر الأصلي] — دفعة واحدة من غير أي اشتراك شهري.
وده شامل: [التركيب + نقل البيانات + التدريب + سنة تحديثات].
العرض ده لأول 20 عميل بس وباقي [X] مقاعد.
```

**خطوة 4 — الإغلاق المباشر (متسألش «إيه رأيك؟»):**
```
تحب نحدد معاد التركيب إمتى — بكرة ولا بعده؟ 📅
```

**لو تردد → روح للاعتراضات تحت. لو وافق:**
```
تمام جدًا 🎉 هحتاج منك:
- العنوان بالظبط
- نوع جهاز الكاشير (أو لو هتجيب جديد نرشحلك)
- لو معاك بيانات قديمة ابعتها من دلوقتي نجهزها
أشوفك [المعاد] إن شاء الله 🤝
```

## 6) بعد البيع

### رسالة ما بعد التركيب (نفس اليوم)
```
مبروك يا [الاسم] 🎉 محلك دلوقتي شغال بنظام ذكي.
تلاقي مرفق: دليل الاستخدام الكامل + ملخص الاختصارات.
أهم 3 حاجات افتكرها:
1️⃣ F8 = بيع كاش وطباعة في ضغطة
2️⃣ الجرس 🔔 بيقولك كل حاجة محتاجة انتباهك
3️⃣ البرنامج بياخد نسخة احتياطية لوحده يوميًا
أي سؤال — أنا رسالة واحدة بعيد عنك 🤝
```

### طلب الترشيح (بعد أسبوعين من التركيب)
```
إزيك يا [الاسم] 😊 عامل إيه مع البرنامج؟
لو مبسوط بيه — أكيد عندك صاحب تاجر زيك يستاهل يريح دماغه:
رشّحه لينا، ولو ركّب هديك [سنة دعم إضافية / خصم على السحابية / مبلغ نقدي] 🎁
ابعتلي رقمه بس والباقي علينا.
```

### طلب فيديو الشهادة (من عميل راضٍ)
```
يا [الاسم]، طلب صغير يفرق معانا جدًا 🙏
ممكن تصور بموبايلك 30 ثانية بس وانت في المحل تقول فيها:
اسمك ونشاطك + بتستخدم ATTARIX من إمتى + أكتر حاجة ريحتك فيه.
الفيديو ده بيساعد تجار زيك ياخدوا القرار — وهديك [هدية/خصم على التحديثات] شكرًا ليك 🎁
```
