🛒

نقطة البيع (كاشير)

  • مسح باركود وبحث فوري بالاسم/الكود
  • شاشة كاشير مرتّبة: الأكثر مبيعًا دايمًا قدامك
  • دفع مختلط: نقدي + بطاقة + تحويل + آجل
  • بيع كاش سريع بمفتاح F8 + طباعة فورية
  • كوبونات خصم ونقاط ولاء وتعليق فواتير
📦

المنتجات والمخزون

  • سعر قطاعي وجملة وشراء وضريبة لكل صنف
  • وحدات متعددة (كرتونة/دستة/قطعة) بباركود خاص
  • متغيرات (مقاس/لون) وطباعة ملصقات باركود
  • تواريخ صلاحية وتشغيلات + تنبيه «بيع الأقرب انتهاءً»
  • تنبيه نواقص حسب حد إعادة الطلب
🏬

المخازن والجرد

  • أكثر من مخزن وأرصدة لكل مخزن
  • جرد فعلي وتسوية الفروقات بضغطة
  • تقييم المخزون بالتكلفة والبيع
👥

العملاء

  • كشف حساب وتحصيل دفعات ونقاط ولاء
  • مستوى سعر تلقائي لكل عميل (قطاعي/جملة)
  • حد ائتمان وتنبيهات مديونية
  • العميل = المورد: كشف موحّد ومقاصة بضغطة
🚚

الموردون والمشتريات

  • فواتير شراء تحدّث المخزون والتكلفة تلقائيًا
  • أرصدة وكشوف حساب وسندات صرف
  • مرتجع مشتريات وإشعارات مدين/دائن
📄

عروض الأسعار والمستندات

  • عرض سعر / أمر بيع / إذن تسليم
  • تحويل المستند لفاتورة بضغطة
  • طباعة A4 أنيقة بشعار محلك و PDF
🧮

محاسبة بقيد مزدوج

  • ترحيل تلقائي لكل العمليات لقيود متطابقة
  • قائمة دخل وميزانية وميزان مراجعة
  • دفتر يومية وأستاذ وشجرة حسابات
  • أوراق قبض ودفع (شيكات) بمتابعة استحقاق

تقارير تفهم منها

  • أرباح فعلية (إيراد − تكلفة) يومي وشهري
  • ربحية كل صنف وكل عميل + تصنيف ABC
  • أعمار ديون العملاء والموردين
  • تقرير ضريبة القيمة المضافة
  • طباعة وتصدير Excel لكل تقرير
🔔

تنبيهات ولوحة تحكم

  • جرس ذكي: نواقص، مديونيات، صلاحيات، شيكات
  • مقارنات تلقائية: الأسبوع ده × اللي فاته
  • بحث شامل Ctrl+K يوصلك لأي حاجة في ثانية
🛡️

مستخدمون وصلاحيات

  • أدوار (مدير/كاشير/مشاهد) بتحكم كامل
  • موافقة مدير على الخصومات الكبيرة
  • سجل عمليات يوثّق كل إجراء (Audit)
💾

نسخ احتياطي وأمان بيانات

  • نسخ تلقائي يومي + نسخ يدوي بضغطة
  • تصدير/استيراد Excel وقاعدة بيانات كاملة
  • تأكيد قبل أي حذف + استرجاع المحذوف (Undo)
  • نقل بياناتك كاملة بين الأجهزة
💵

الكاش والمصاريف

  • وردية كاشير (فتح/غلق الدرج) وحساب الفروقات
  • خزائن وحسابات نقدية وبنكية
  • مصاريف بالبنود مربوطة بالخزينة

أسئلة شائعة

كل اللي بيسألوه قبل الشراء

محتاج إنترنت عشان يشتغل؟

لأ نهائيًا. النسخة المكتبية بتشتغل 100% بدون إنترنت وبياناتك على جهازك أنت. الإنترنت بس لو حبيت تبعت تذكيرات واتساب أو تستخدم النسخة السحابية.

لو الجهاز باظ… بياناتي بتضيع؟

لأ. فيه نسخ احتياطي تلقائي يومي + تصدير قاعدة البيانات كاملة بضغطة، تنقلها لأي جهاز تاني وتكمل شغل عادي. وكمان أي حذف بالغلط جواه ليه زر «تراجع» يرجّعه.

أنا شغال بدفاتر أو ببرنامج قديم — هنقل البيانات إزاي؟

في الباقة الاحترافية بننقلك الأصناف والعملاء والأرصدة بنفسنا من النظام القديم أو من ملفات Excel — بتفتح البرنامج تلاقي محلك جاهز.

الكاشير عندي مش متعود على الكمبيوتر…

الشاشة مصممة زي الكاشير الحقيقي: مسح باركود وزر واحد للبيع الكاش. وبندرّبه بنفسنا في الباقة الاحترافية، وفيه دليل اختصارات جوه البرنامج بضغطة ؟

فيه اشتراك شهري؟

النسخة المكتبية دفعة واحدة وبدون أي اشتراك إجباري. النسخة السحابية باشتراك مريح لأنها شاملة سيرفر وتحديثات مستمرة.

هل يدعم الضريبة والفواتير الضريبية؟

أيوه — ضريبة قيمة مضافة اختيارية لكل فاتورة، رقم ضريبي على المطبوعات، وتقرير ضريبة كامل (مخرجات ومدخلات وصافي).
